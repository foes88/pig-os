05 Core Feature

AI Agent Design

Agent 1 — Loss Detection

Rule-based, No benchmark, Day 1

NPD Loss = (today - weaning - 5d) × daily_cost Delay Loss = (today - mating - 114d) × cost × 1.5
이 농장에서 놓치는 돈: ₩2,688,000
NPD 8두 ₩2,016K + 분만지연 2두 ₩672K

Agent 2 — Action Scheduler

Auto-calendar + Push notifications

Mating Input +25d → Preg Check (D-3 alert) +114d → Farrowing (D-1 alert) +21d → Weaning (D-1 alert) +5d → Re-mate (D+0 alert)
Today's Actions (3)
#042 Farrow D-1 | #018 Re-mate | #055 Preg Check

Agent 3 — Anomaly Detection

Self-baseline, No ML, MVP+2wk

Feed: current > avg × 1.20 or < 0.80 Weight: current < expected × 0.80 Mortality: current > avg × 1.50
90-day baseline comparison
Phase 2: Federated Learning cross-farm
87.5% of AI features work Day 1 without benchmarks
