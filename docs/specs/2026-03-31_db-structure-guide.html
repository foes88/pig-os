Design Principles

Schema-per-TenantEach farm gets isolated schema. farm_id on every table for RLS fallback.
Soft Deletedeleted_at TIMESTAMPTZ on all operational tables. Never hard delete.
Audit TrailAll CUD operations logged to audit_log with old/new JSONB.
Offline Syncsync_queue + offline_created_at on event tables. LWW conflict resolution.
Period LockMonth-end closing via period_locks. Locked periods reject INSERT/UPDATE.
Event-Driven StatusSow status auto-transitions via triggers on event INSERT (mating/farrowing/weaning).
i18nCode tables (code_groups/values/translations) for all enumerated values.

NPD 6 Components

Non-Productive Days Breakdown (Global Standard)

Gilt NPD (후보돈)
NPD-01 Entry → 1st Mating
avg ~160 days | v_npd_gilt_entry

NPD-02 Entry → Removal (no mating)
100% wasted | v_npd_gilt_removal
Sow NPD (경산돈)
NPD-03 Weaning → 1st Mating (WEI)
target 4-7 days | v_sow_npd

NPD-04 Weaning → Removal (no mating)
v_npd_wean_to_removal

NPD-05 Service → Re-Service (preg fail)
~21d regular / ~42d double | v_npd_reservice

NPD-06 Service → Removal (preg fail + cull)
v_npd_service_to_removal
Economic Impact: NPD 1 day = $3.24 USD (US) / ₩8,500 KRW (KR)
Top herds: ~36 NPD/year | Bottom herds: ~72 NPD/year | Gap = ~$117/sow/year
